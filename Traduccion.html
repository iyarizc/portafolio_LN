📝 Código del Ejemplo

# Importamos la librería para traducción
from deep_translator import GoogleTranslator

texto="hola mundo"

ingles=GoogleTranslator(source='es', target='en').translate(texto)
frances=GoogleTranslator(source='es', target='fr').translate(texto)
aleman=GoogleTranslator(source='es', target='de').translate(texto)
arabe=GoogleTranslator(source='es', target='ar').translate(texto)

print("Ingles: ",ingles)
print("Frances: ",frances)
print("Aleman: ",aleman)
print("Arabe: ",arabe)

📚 Glosario de Funciones

1. GoogleTranslator(source='es', target='en').translate(texto)

Esta funcion traduce el texto a el idioma seleccionado.
